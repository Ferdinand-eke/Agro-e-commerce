To set up and use this project

1. download from github directory

2. import project folder "TradeAfrique" into netbeans IDE

3. creare database with name "agro_trade_afrique"  .

4. import .sql file from foldernamed db into xampp.

5. Open project in netbeans and follow the path to the sql connection file

		Sorce Packages > conn > MySQLConnUtils : 
			1. change userName = " NB: your own xampp mysql username "; mine here is "root"
			2. change password = " NB: your own xampp mysql password";  mine here is "ferazi"

So with this settings in place, you are set to use the project on your local machine. Thanks!
